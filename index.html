Bootstrap Elements

Quick index of component pages in this folder
About these components

These example component pages are built with Bootstrap and simple HTML/CSS. You can open any page, copy the HTML and CSS from it, and paste into your project.

  • Bootstrap version used on this index: Bootstrap 5.3.2 (CDN)
  • Optional theme file: barbar.css (if present)
  • No JavaScript is required on this index — links are plain anchors to the example pages.
